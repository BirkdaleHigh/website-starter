Attributes

Read along with the source code open

Like Superman has an attribute of the ability to fly, so can tags get attributes.

As alwyas there are rules that must be followed;

  1. Attribute values must be wrapped in quotes: " "
  2. An attribute can be set to something with =
  3. Attributes must be after the tag name inside the tags wrapping symbols.

These few simple rules define the Language part of HTML

Best guidelines to follow

  • Write all attributes in lower case.
  • Write all values in lower case.

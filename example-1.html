HTML Has rules that must be followed;

  1. All tags are wrapped in < and > symbols
  2. Opening tags must look like: <name>
  3. Closing tags must look like <name/>
  4. Tags that are opened must be closed.
  5. Opened tags must be closed in the same order. This is called Nesting.

These few simple rules define the Language part of HTML

Best guidelines to follow

  • Write all tags in lower case.
  • Clearly separate tags with new lines.
  • Indent tags as they get nested.
